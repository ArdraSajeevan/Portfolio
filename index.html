SkillBridge

Built with Python Flask

SkillBridge is a web-based platform that connects daily wage workers with employers based on required skills and job duration. The system allows employers to post jobs and workers to apply, with real-time notifications, JWT-based authentication, and efficient job management. Built using Python Flask and MongoDB, it focuses on empowering local employment without relying on location APIs.

Two Pass Assembler

Built with Java

Developed a Two-Pass Assembler in Java with a user-friendly GUI using Swing. The application reads assembly code, constructs symbol and literal tables in Pass 1, and generates machine code in Pass 2, handling forward references and intermediate code generation. The GUI allows users to load files, view tables, and track the translation process in real-time.
